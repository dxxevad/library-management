# 📚 LibraTrack — Library Management System

A modern, full-featured library management system built with vanilla HTML, CSS, and JavaScript.

## 🚀 Features

- **Dashboard** — Real-time stats (total books, members, borrowed, overdue)
- **Book Catalog** — Add, search, and delete books with availability tracking
- **Member Management** — Register and manage library members
- **Borrow & Return** — Issue books to members, track due dates, mark returns
- **Records** — Full history of all borrow transactions with status (Active / Overdue / Returned)
- **Activity Feed** — Live log of recent actions
- **Local Storage** — All data persists in the browser

## 🛠️ Tech Stack

- HTML5, CSS3, Vanilla JavaScript
- Google Fonts (Playfair Display + DM Sans)
- LocalStorage API for data persistence
- No frameworks, no dependencies

## 🖥️ How to Run

1. Clone or download this repository
2. Open `index.html` in any browser
3. That's it! No server needed.

```bash
git clone https://github.com/YOUR_USERNAME/library-management.git
cd library-management
# Open index.html in your browser
```
